#Snake Game
My first Python game built using Pygame.

## Features
- Snake movement with arrow keys
- Food spawning and growth
- Collision detection (walls and self)
- Game over screen
- Restart option (press R)

## How to Run
1. Install Python and Pygame
2. Run:
   python snake.py
